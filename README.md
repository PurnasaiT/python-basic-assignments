# Python Beginner Programs

This repository contains simple Python scripts created while learning the basics of programming.  
They cover user input, string handling, and basic math operations.

## Programs Included

### 1. Basic Math Operations
- Performs addition, subtraction, multiplication, and division.
- Takes two numbers as input from the user.
- Displays the results of each operation.

### 1.2 Greeting Script
- Asks for your first and last name.
- Combines them into a full name.
- Prints a personalized welcome message.
